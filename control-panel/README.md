# OpenClaw Control Panel

React-based control panel for managing OpenClaw AI Platform.

## Tech Stack

- React 18 + TypeScript + Vite
- shadcn/ui + Tailwind CSS
- TanStack Query + Zustand
- TanStack Router

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Features

- Dashboard: Service status and resource monitoring
- Service Management: Start/stop/restart services
- Configuration Center: Manage models, API keys, environment variables
- User Management: Authentication and authorization
